# QR_generator
A lightweight and simple QR code generator.

All user requests are processed directly on the client and are not stored on the server.

## How to use

### Online
Just visit https://qr.sufl.cat

### Offline
1. Download the [source code](https://github.com/SuFl3770/QR_generator/releases/tag/release).
2. Run index.html.
